# MarkDown语法记录：
1. 多个#+空格+内容 为标题，有几个#就是几级标题。

2. 利用"**"来选中强调的内容。
（eg：**这是被强调的内容**）

3. *利用"*"*来选中斜体的内容*

4. 用"~~"来选中需要画删除线的内容
(eg:~~删除线内容~~)

5. 用"=="选中要高亮的内容
(eg:==这是高亮的内容==)

6. 用`圈中要写的代码, 用```圈中要写的代码块

7. 用>+空格表示引用
> 这是一个引用

8. 无序列表用"-"+空格

eg:
- abc
- def

9. 上标用^包围，下标用~包围
(eg:H~2~O, 10^5^)

10. 用[]包围文本, 用()包围链接
11. 任务列表: -+空格+[ ]+空格

eg:
To Do List:
- [ ] 1
- [x] 2
